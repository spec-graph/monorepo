# Task Decomposition Pack Context

## When This Pack Activates
- Always (every project needs task decomposition)

## What This Pack Provides
- Epic template
- Story template (with INVEST checklist)
- Story slicing validation gate

## Key Artifacts
- `plan/epic` — Large feature grouping
- `plan/story` — Implementable user story

## Agent Guidance
- Stories must follow INVEST: Independent, Negotiable, Valuable, Estimable, Small, Testable
- Each story must have at least 2 acceptance criteria
- Stories should be implementable in 1-3 days
- Epics group related stories by user activity
